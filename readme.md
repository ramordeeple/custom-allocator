# Кастомный аллокатор памяти 

## Описание

Проект реализует кастомный аллокатор памяти на языке C для фиксированных блоков.
Поддерживаются только два размера блоков:

* Малые блоки: 15 байт
* Большие блоки: 180 байт

Реализуются функции:

* `malloc(size_t size)` — выделение блока памяти
* `free(void* ptr)` — освобождение ранее выделенного блока

Аллокатор использует **пулы блоков** и поддерживает быстрые операции O(1).

Подходит для 8/16/32-битных архитектур.

* **allocator.h** — прототипы функций и константы (размеры блоков, размеры пулов).
* **allocator.c** — реализация пулов, malloc/free, ленивой инициализации.
* **test_allocator.c** — тесты всех функций аллокатора, проверка корректности работы.

## Тесты

Тестовый файл `test_allocator.c` проверяет:

1. Выделение и освобождение малых блоков (`test_small_blocks`)
2. Выделение и освобождение больших блоков (`test_large_blocks`)
3. Корректность записи и чтения данных (`test_data_integrity`)
4. Повторное выделение после `free` (`test_free_and_realloc`)
5. Поведение при исчерпании памяти (`test_exhaustion`)

Все тесты выводят лог и завершаются с `EXIT_SUCCESS` или `EXIT_FAILURE`.


## Особенности реализации

* **Фиксированные блоки** для скорости и предсказуемости
* **Ленивая инициализация пулов** при первом вызове `malloc`
* **O(1) операции** malloc/free
* **Защита от некорректных указателей**: `free` игнорирует чужие указатели
* Совместимость с 8/16/32-битными архитектурами
